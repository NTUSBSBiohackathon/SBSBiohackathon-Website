101 Introductory Python Workshop
Data: 15 Jan 2020 (Wednesday)
Time: 1830 - 2130 (Inclusive Time Break)
Location: SBS TR-5 (Subject to changes)

Basic workshop that is for student without any knowledge of programming language and python. This basic workshop is for beginners that will give students the skills that they need to tackle common biological and bioinformatics problems that they require for projects or research work. Moreover, it will help students who are going to take BS1009 Introduction to Computational Thinking. We will also link it to Biology!

Sign Up
  • Variables and data types
  • Input function and data type conversion
  • Math Functions
  • Strings (Concatenation and Accessing and slicing)
  • Python statement
  • Volume of Rectangular Prism, BMI Calculator Genetic Code storage and cGPA Calculator
102 Data Analysis with pandas
Data: 22 Jan 2020 (Wednesday)
Time: 1830 - 2130 (Inclusive Time Break)
Location: SBS TR-5 (Subject to changes)

Pandas is a Python library that can make data analysis much simpler. In this tutorial, we'll use Python and pandas to analyze healthcare-related data from data.gov.sg/. We will use real live example by analyzing the number of doctors and hospital beds against the number of admitted patients using the information for the data.gov.sg. Thus, able to see if Healthcare sector is sufficiently equipped to support the number of admitted patients.

Sign Up
  • How to import library (Numpy and Pandas)
  • Series
  • Instantiate a DataFrame from an external CSV file
  • Dataframe function (.shape, .dtypes, .columns, .head, .samples)
  • Filtering
  • Univariate analysis and Summary Statistics

Pricing and Deposit-Refund System

Pricing: SGD$4.00 per person per workshop.

Certification
Certification and Professional Certification

Certificate with Credential ID and Credential URL will be given out signed by chairman and vice-chairman of Biohackathon Club. Students who attended at least 2 workshops and hackathon will be given professional certifcation at the end of the hackathon events signed by Dr Wilson Goh. Only E-Certifcation will be given for all workshops.

For student who want to access their certifcation, can type sbsbiohackathon.com/Certifcation/Matric No/1

Example, https://sbsbiohackathon.com/Certification/U1500001A/1

Hackathon

This hackathon encourages undergraduate students to come up with creative ideas to solve the current problems in the life science field using programming approaches. It will be a 1-week event, where the 1st three days will be a workshop or students to learn programming skills that are required for the challenges. The fourth and fifth week of the event, students will come up with ideas to deal with the following challenges stated below.

Workshop will be conducted by Prof from LKC School of Medicine and School of Biological Sciences. Each of these workshops will help them to come with ideas and solve problems.

Data: 11 May 2020 – 15 May 2020
Price: Free
Location: CR2 or 3 (Subject to changes)
